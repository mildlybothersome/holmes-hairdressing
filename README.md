# holmes-hairdressing

Holmes Hairdressing is an independant hairdressing salon and in this project I will be creating a website for them.

The website will be a 3 page responsive site with a mobile focus approach as this is the most used format clients will contact the stylist.
These pages will be a Home page with client testimonials, a Gallery with previous work and a contact page. 
The site will contain colours corresponding to the established brand.
